# Tribulnation Labs — Barcelona

Crypto trading firm. We build the tools we trade on — and make them open source.

---

### [Litmus Ledger](https://litmus.tax)

Your crypto data is probably wrong. Exchange exports are inconsistent, timestamps drift, cost basis is miscalculated. Litmus runs your data through a rigorous validity test, diagnoses exactly what's broken, and produces auditable corrections you can approve.

Early access at [litmus.tax](https://litmus.tax).

---

### [Typed](https://tribulnation.com/typed)

Async, validated Python clients for every exchange we trade on. One package per venue, under the `typed-*` PyPI namespace.

```bash
pip install typed-hyperliquid
pip install typed-mexc
pip install typed-alchemy
```

### [SDK](https://github.com/tribulnation/sdk)

One interface across every venue. Write your strategy once, swap the exchange by changing a string. Built on top of Typed.
